Твой Ferrrit 3.1.0

В этом обновлении расширен и улучшен функционал приложения для сервисных специалистов Ferrit.

Основные изменения:
- добавлен раздел кодов ошибок Ferrit с быстрым поиском по номеру;
- улучшены главный экран и навигация по основным разделам;
- добавлены дополнительные настройки отображения интерфейса;
- повышена плавность работы приложения и удобство использования;
- доработана логика сохранения состояния и продолжения работы;
- исправлены выявленные ошибки и повышена общая стабильность;
- добавлена пасхалка.

Приложение продолжает развиваться как удобный оффлайн-инструмент для диагностики, поиска неисправностей, работы со схемами и инженерной поддержки в полевых условиях.
