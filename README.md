# Твой Ferrrit

Оффлайн-инструмент для сервисных специалистов Ferrit.

Приложение собрано как рабочий полевой набор для диагностики, поиска неисправностей, просмотра электрических схем и быстрого доступа к инженерной справочной информации без подключения к интернету.

## Возможности

- оффлайн-доступ к PDF-схемам и сервисным материалам;
- поиск по симптомам, кодам ошибок Ferrit и кодам двигателя;
- раздел датчиков и инженерных калькуляторов;
- быстрый переход по сериям техники Ferrit;
- сохранение последних открытий и рабочих сценариев;
- настройки интерфейса для удобной работы в поле;
- встроенная пасхалка.

## Сборка

Основная релизная сборка:

```powershell
flutter build apk --release --obfuscate --split-debug-info=build\app\outputs\symbols
```

Путь к APK после сборки:

```text
build\app\outputs\flutter-apk\app-release.apk
```
