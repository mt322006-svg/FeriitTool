Твой Ferrrit 3.0.1
Автономный инструмент для сервисной диагностики техники Ferrit.

Что нового
обновлены главный экран и сервисная навигация;
добавлены настройки:
увеличенный текст;
не выключать экран при просмотре схем;
светлая и тёмная темы;
скрытие/показ быстрых инструментов;
улучшен просмотр PDF:
запоминание последней страницы и масштаба;
кэш при открытии PDF;
поиск по тексту;
более удобное листание;
добавлены карты технического обслуживания по моделям;
переработан раздел кодов двигателей Cummins:
отдельный поиск по Fault / SPN / FMI;
поиск по описанию;
выбранные коды;
копирование кода;
подсказки «что посмотреть в первую очередь»;
добавлены датчики и калькуляторы;
расширены возможности устранения неполадок в ПДМ-17;
добавлена методика проверки CAN для ПДМ-14 / ПДМ-17 / ШС-30.
Важно
приложение работает без подключения к интернету и содержит встроенные PDF-схемы;
данные по некоторым сценариям еще будут уточняться на основе реальной сервисной практики;
для CAN на этих машинах ориентируйтесь на сопротивление линии: около 120–130 Ом при обесточенной машине.
Установка
установить APK на Android-устройство;
при необходимости разрешить установку из файла;
если на устройстве уже установлена старая сборка с другой подписью, перед установкой ее может потребоваться удалить.
