# Твой Ferrrit 3.0.1

Оффлайн-инструмент для сервисной диагностики техники Ferrit.

## Что нового

- обновлён главный экран и сервисная навигация;
- добавлены настройки:
  - увеличенный текст;
  - не выключать экран при просмотре схем;
  - светлая и тёмная темы;
  - скрытие/показ быстрых инструментов;
- улучшен просмотр PDF:
  - запоминание последней страницы и масштаба;
  - кэш открытия PDF;
  - поиск по тексту;
  - более удобное листание;
- добавлены карты ТО по моделям;
- переработан раздел кодов двигателя Cummins:
  - отдельный поиск по Fault / SPN / FMI;
  - поиск по описанию;
  - избранные коды;
  - копирование кода;
  - подсказки "что смотреть первым делом";
- добавлены датчики и калькуляторы;
- расширен troubleshooting по ПДМ-17;
- добавлена методика проверки CAN для ПДМ-14 / ПДМ-17 / ШС-30.

## Важно

- приложение оффлайн и содержит встроенные PDF-схемы;
- для части сценариев данные ещё будут уточняться по реальной сервисной практике;
- для CAN на этих машинах ориентир по сопротивлению линии: около 120–130 Ом на обесточенной машине.

## Установка

- установить APK на Android-устройство;
- при необходимости разрешить установку из файла;
- если на устройстве уже стоит старая сборка с другой подписью, её может потребоваться удалить перед установкой.
