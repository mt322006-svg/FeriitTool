# Правила траблшутинга

## Формат симптома
Симптом = коротко + контекст (модель/узел)
Пример:
- "DNK10: нет питания"
- "DNK17: фары стрела низ не горят"

## Формат решения (шаги)
Шаг = действие + точка измерения + ожидаемый результат.
Пример:
1) Измерить +24V на <разъём/пин>
2) Если нет → идти в ветку A
3) Если да → ветка B

## Запреты
- Нельзя добавлять разъёмы/пины/номера "по опыту"
- Нельзя писать “проверь всё”
- Можно писать общие проверки только если они не требуют цифр (например “осмотр жгута”), но любые цифры/пины — только из схем.
