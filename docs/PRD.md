# Ferrit Tool — PRD

## 0. Суть
Ferrit Tool — офлайн-инструмент для диагностики и быстрых проверок по технике Ferrit (DNK10 / DNK14 / DNK17) + встроенный просмотр схем (PDF).

## 1. Пользователь
- Электрик/диагност/механик на выезде или в цеху.
- Нужна скорость, офлайн, без лишних экранов.

## 2. Главные правила
- Offline-first: всё основное без интернета.
- Без выдумывания: разъёмы/пины/номера — только из схем. Если данных нет — так и пишем.
- 2–3 тапа до действия.
- UI строгий (industrial): графит, светло-серый на сером; акцент — янтарь/оранж только для важных элементов.

## 3. Навигация (v0.1)
Home → выбор модели (DNK10/14/17)
→ список узлов/категорий (электрика и т.п.)
→ список симптомов
→ экран симптома: чек-лист + кнопка "Открыть схему"
→ PDF viewer (внутри приложения)

## 4. Функции v0.1
- Список моделей: DNK10 / DNK14 / DNK17
- Симптомы (минимум):
  - Нет питания
  - Нет света (стрела/низ/верх — если есть данные)
  - Нет зарядки
- Экран симптома:
  - чек-лист (отмечаем “проверено”)
  - кнопка "Открыть схему"
- PDF viewer:
  - pinch-zoom
  - swipe
  - быстрый переход по страницам (страница/слайдер/ввод/список)
- Хранилище PDF: assets/pdfs/
- Индексация страниц: assets/index/pdf_index.json

## 5. Не делаем в v0.1
- Поиск по тексту схем
- OCR
- Онлайн-бот/сервер
- Автогенерация пинов из картинки
