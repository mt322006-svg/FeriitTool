# Схемы (PDF)

## Папка
assets/pdfs/
- dnk10_schema.pdf
- dnk14_schema.pdf
- dnk17_schema.pdf

## Привязка
Привязка делается через:
- модель → путь к pdf
- модель → индекс страниц (assets/index/pdf_index.json)

## Обновление
При обновлении схем:
1) заменить PDF в assets/pdfs/
2) проверить номера страниц
3) обновить pdf_index.json
4) прогнать руками: открыть схему → jumpToPage на 3–5 ключевых узлов
